## 📌 项目简介

**黑墓杯**是一套以圣杯战争为核心的TRPG规则系统。玩家扮演御主（Master）与从者（Servant），在虚构城市中展开情报战、战术对抗与角色扮演，争夺万能的许愿机——圣杯。

本规则起源于《黑幕圣杯规则》，经过大量重构、扩展与平衡调整，已形成独立的规则体系，并持续更新中。

## 🎯 核心特色

- **自由构筑**  
  从者：使用金方块自由分配能力值、技能、宝具、武装，支持常规7职阶 + 多种特殊职阶（Ruler、Avenger、Foreigner 等）及特殊模版（亚从者、凭依从者、概念从者、多体从者等）。  
  御主：使用绿方块构建背景（天选之人 / 武道 / 财富 / 魔道 / 教会 / 知识 / 军武）及扩展模版（死徒、魔女、孪生子等）。

- **策略战斗**  
  红（猛击）> 蓝（技击）> 绿（迅击）> 红的克制循环，配合预知值、暴击星、硬直、连续指令、宝具对轰、距离系统（近 / 中 / 远 / 超距），实现格斗游戏般的心理博弈。

- **情报战**  
  暴露值、侦查值、魔力波动、线索、真名推测、气息遮断……情报决定胜负。

- **完整世界观**  
  令咒（多种用法）、灵脉（吸收大源）、抑止力（阿赖耶 / 盖亚）、补魔、阵地建造、道具制作、圣杯强化、海战 / 空战 / 超距战等。

- **持续更新**  
  从 0.2 到 3.2，历经 60+ 个版本迭代，并配有《魔术万能攻略书》《NPC阵营扩展书》等扩展资源。

## 🚀 快速上手

1. **获取规则书**：进入 `规则书/` 目录，下载最新版 `黑墓杯3.2.pdf`
2. **阅读顺序**（新手建议）：  
   御主教程 → RP指南 → 战斗入门 → 从者/御主构筑规则
3. **车卡**：使用 `身份卡/` 目录下的空白卡模板（从者卡 / 御主卡）
4. **开团**：GM 需准备模组地图、NPC、初始剧情与审卡标准（可参考 `NPC阵营拓展/`）
5. **跑团平台**：建议使用 QQ群 + 骰子机器人（如塔罗牌、Dice+）
